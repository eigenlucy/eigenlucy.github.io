Bluetooth/FM Transmitter

DIY SI4712/ESP32 Base Bluetooth->FM Transmitter

This is an FM transmitter project based on the Silicon Labs SI4713. It uses an ESP32 microcontroller to receive bluetooth audio via A2DP using the ESP32-A2DP Audio Library. This project was designed in AUTODESK EAGLE/Fusion360. Here is a GitHub repo containing the PCB files, codebase, and BOM.
